# Samsung Innovation Campus — Modül 1 / Lab 2

Kişisel marka web sitesi: tek dosyalık (HTML + CSS + JavaScript) modern ve responsive bir tanıtım sayfası.

## İçerik

- **Hero:** Ad, unvan (Makine Mühendisi | Yapay Zeka), slogan ve navigasyon
- **Ben Kimim** — tanıtım bölümü
- **Yeteneklerim** — teknik beceriler bölümü
- **Gelecek Hedefim** — kariyer hedefleri bölümü
- **Footer:** E-posta ve LinkedIn bağlantıları

## Kullanılan Teknolojiler

- Saf HTML, CSS ve JavaScript (framework yok, tek dosya: `index.html`)
- Google Fonts (Inter)
- IntersectionObserver ile kaydırma animasyonları, smooth scroll

## Çalıştırma

`index.html` dosyasını tarayıcıda açmanız yeterli.
